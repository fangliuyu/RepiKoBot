.duel/.duel!/.duel！/.决斗/.决斗！/.打牌/.打牌！/.牌/.牌！/.房
  ygo约战开房！
  产生房间代码，主要用于233服
  可以记录生成的房间代码，方便之后快捷调出
  详见 .help duel detail
例：
  .duel 房 -t -tm 0  
      => TM0,T#房
  .duel 房 -麻将 -时间 0  
      => TM0,T#房（同上一条）
  .duel 俺的房 -set  
      => 记录了房间【俺的房】
  .duel -me  
      => 俺的房
  .duel  
      => 记录后同上一条
  .duel -23333  
      => 23333端口
  .duel -s 编年史  
      => 决斗编年史
